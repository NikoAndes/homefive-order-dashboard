# Roadmap

## Version 1.0 (Current)

- [x] Static HTML/CSS/JS dashboard
- [x] 8 status metric cards
- [x] Filterable orders table
- [x] Search by customer, city, or order ID
- [x] Responsive design
- [x] 30 fictional orders dataset
- [x] Dark theme design system

## Version 2.0 (Planned)

- [ ] Node.js/Express REST API backend
- [ ] User authentication and roles
- [ ] PostgreSQL database
- [ ] Create and edit orders from the UI
- [ ] Date range filters
- [ ] Export to Excel / CSV

## Version 3.0 (Future Vision)

- [ ] Shopify orders webhook integration
- [ ] Real-time carrier tracking (Servientrega, TCC, Coordinadora)
- [ ] Analytics charts (revenue, delivery rate)
- [ ] PDF invoice generation
- [ ] Multi-store support
- [ ] Mobile app (React Native)
